## Contributing

### Demo server

```
npm run demo-app
```

### Build

Build the library into `dist/lib` and copy the library into `node_modules` for the demo app
to use.

```
npm run build && npm run copylib
```

### Watching

```
npm run watch
```

### Testing

Run karma tests

```
npm run test
```

To run the tests just once

```
npm run test:once
```
