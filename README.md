# SatAttachedOverlay

## Building

```
yarn build
```

## Watching

```
yarn watch
```

## Demo server

```
yarn start
```

## Other

Built with  [Angular CLI](https://github.com/angular/angular-cli) and
[ng-packagr](https://github.com/dherges/ng-packagr).
